## Adicionar um produto
Acesse o **menu principal**, em seguida clique em **Produtos/Serviços > Produtos**. 

![Aba produtos](@site/static/img/cadastrar-produtos/aba-produtos.png)

Será mostrado uma tabela com todos os produtos cadastrados.

Na barra de ferramentas no canto superior direito da tabela clique no ícone **Ações** ![Icone ações](@site/static/img/cadastrar-produtos/icone-acoes.png) e então **Adicionar Produto**.

Serão apresentados os campos necessários para cadastro do produto.

- **Código**: Determina qual será o código de identificação do produto para controle interno, ele será automaticamente incrementado em 1 a cada novo produto registrado.

Pode ser deixado como padrão.

- **Descrição**: O nome do produto.

- **Prazo de validade**: Data de validade do produto. Deve ser preenchido caso o produto apresente característica perecível.

- **Tipo de Produto/Código de Barras**: descrição
	
  - Produto simples [Preço]: Para produtos especificados apenas pelo preço
  - Produto simples [Margem]: Para produtos especificados apenas pela margem
  - Produto simples com tabelas de pesquisa [Preço]:  ___
  - Produto simples com tabelas de pesquisa [Margem]:  ___
  - Balança/Peso EAN (KG) [Preço]:  ___
  - Balança/Peso EAN (KG) [Margem]:  ___
  - Balança/Preço [Preço]:  ___
  - Código + ID de Grade [Preço]:  ___
  - Código = ID de Grade [Preço]:  ___
  - Produto com opção [Preço Venda + Opção]:  ___
  - Produto com opções [Preço Venda + Opções]:  ___
  - Desdobramento de produto:  ___
  - Produto de Tabela [Preço]:  ___
  - Produto de Tabela [Margem]:  ___

- **Composição**: Especifique o tipo do produto
  - Produto Simples
  - Produto Composto [venda do produto]
  - Produto Composto [venda da composição]

**Código de Barras**: O código de barras atribuído ao produto.

Selecione a modalidade da operação
  - Compra: Se o produto está sendo comprado
  - Venda: Se o produto está sendo vendido
  - PDV: Se o produto está sendo vendido por meio de um PDV
